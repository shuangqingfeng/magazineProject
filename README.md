###### 项目用到了如下技术：
- 项目结构：根据 google 提供的范例[todo-mvp](https://github.com/googlesamples/android-architecture/tree/todo-mvp/) 
- 网络框架：rxjava + retrofit
- 多类型数据适配器使用了 drakeet 的 [multitype](https://github.com/drakeet/MultiType)
- 注解： butterkinfe
